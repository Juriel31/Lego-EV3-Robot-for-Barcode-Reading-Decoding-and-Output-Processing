## LEGO EV3 Barcode Reading Robot

This project implements a LEGO EV3 robot capable of reading barcodes, decoding them using Hamming code logic, and outputting the processed data. The robot uses a light sensor to detect barcode patterns and a stack-based preprocessing algorithm for accurate decoding.

### Features
- Barcode Reading: Uses LEGO EV3 light sensor to detect black/white patterns.
- Hamming Code Decoding: Detects and corrects errors in scanned barcodes.
- Stack-Based Preprocessing: Organizes scanned data for decoding.
- Output Display: Prints/Speaks decoded information in real-time.
- Educational Platform: Built using LEGO EV3 kit for hands-on learning in robotics, pattern recognition, and embedded control.
